# Diagnostico

### Objetivos
- Identificar y diagnosticar problemas comunes en programas de Python.
- Analizar y optimizar el rendimiento de scripts para mejorar su eficiencia.

### Contenido
En esta lección, se profundizará en la identificación y resolución de problemas frecuentes que pueden surgir durante la ejecución de código en Python. Se abordarán aspectos como errores sintácticos, excepciones no manejadas y problemas de lógica. También se explorará cómo utilizar herramientas integradas del propio intérprete para diagnosticar estos problemas, como el uso de `print()` y la consola de depuración.

Además, se discutirá la importancia de medir el rendimiento de los scripts a través de métodos como el cálculo de tiempo de ejecución con `timeit` y el análisis de la complejidad del algoritmo. Se proporcionarán ejemplos prácticos para entender cómo optimizar el código mediante técnicas como la eliminación de bucles innecesarios, la utilización eficiente de listas y diccionarios, y la aplicación de algoritmos más eficientes.

### Ejercicio
Implementa un programa que calcule los primeros 100 números primos. Identifica posibles problemas en el código proporcionado y utiliza `timeit` para medir su rendimiento inicial. Luego, optimiza tu implementación y compara nuevamente los tiempos de ejecución.

### Resumen
- Se aprendió a diagnosticar errores comunes en Python.
- Se destacó la importancia del análisis de rendimiento y las herramientas disponibles para ello.
- Se enfatizó la necesidad de optimizar el código para mejorar su eficiencia.
