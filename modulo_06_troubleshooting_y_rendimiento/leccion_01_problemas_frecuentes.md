# Problemas frecuentes

## Contenido
En esta lección sobre problemas frecuentes en Python, se abordarán errores comunes que pueden surgir durante la programación y cómo resolverlos. Los temas clave incluirán manejo de excepciones, optimización del código para mejorar el rendimiento y técnicas para depurar programas. Se destacará la importancia de entender las causas subyacentes de los problemas para evitar errores recurrentes en futuros proyectos.

## Ejercicio
Escribe un programa simple que calcule la media de una lista de números. Identifica y corrige al menos tres posibles errores comunes en tu código, utilizando el manejo de excepciones adecuado y técnicas de depuración básicas.

## Resumen
- Se destacará la importancia del manejo de excepciones para gestionar errores de manera eficiente.
- Se enseñarán técnicas de optimización codificación para mejorar el rendimiento del programa.
- Se practicará la identificación y corrección de errores comunes a través de un ejercicio práctico.
