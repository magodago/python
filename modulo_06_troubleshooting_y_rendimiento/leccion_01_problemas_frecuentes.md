# Problemas frecuentes

## Contenido
En esta lección sobre problemas frecuentes de Python, se abordarán errores comunes que los programadores pueden encontrar al trabajar con la sintaxis y las estructuras básicas del lenguaje. Se discutirá cómo identificar y solucionar problemas relacionados con variables no definidas, tipos de datos incorrectos, bucles infinitos y excepciones no manejadas. El objetivo es proporcionar a los estudiantes herramientas para mejorar el rendimiento de sus programas al diagnosticar y corregir estos errores de manera eficiente.

## Ejercicio
Implemente un programa que calcule la media de una lista de números enteros. Asegúrese de manejar correctamente las excepciones en caso de que se introduzcan valores no numéricos. Prueba tu código con diferentes listas, incluyendo casos donde los elementos no son números y observa cómo el programa responde.

## Resumen
- Se destacarán errores comunes como variables no definidas y tipos de datos incorrectos.
- Los estudiantes aprenderán a manejar excepciones para mejorar la robustez del código.
- El ejercicio práctico ayudará a los estudiantes a aplicar estos conocimientos en una situación real.
