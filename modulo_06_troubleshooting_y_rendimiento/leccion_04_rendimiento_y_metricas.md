# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas de Python, se abordarán los aspectos más importantes para evaluar la eficiencia del código escrito en Python. Se aprenderá a utilizar herramientas como `timeit` y `cProfile` para medir el tiempo de ejecución y analizar el rendimiento de funciones y programas. Además, se discutirá cómo optimizar el código mediante técnicas como la gestión eficiente de memoria y la utilización de estructuras de datos apropiadas.

## Ejercicio
Escribe un programa en Python que calcule los primeros 100 números de Fibonacci utilizando dos métodos diferentes: recursión y iteración. Utiliza `timeit` para medir el tiempo de ejecución de cada método y compara sus rendimientos. Luego, utiliza `cProfile` para analizar la ejecución del programa con el método recursivo y identifica posibles áreas de optimización.

## Resumen
- Se aprendió a utilizar herramientas como `timeit` y `cProfile` para medir y analizar el rendimiento de código en Python.
- Se discutió la importancia de la optimización del código, incluyendo la gestión eficiente de memoria y la selección adecuada de estructuras de datos.
