# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas de Python, se explorará cómo medir y optimizar la eficiencia del código. Se aprenderá a utilizar herramientas como `timeit` para medir el tiempo de ejecución de funciones y `cProfile` para analizar el rendimiento en profundidad. Además, se discutirá la importancia de seleccionar las estructuras de datos adecuadas y optimizar bucles para mejorar la velocidad de ejecución del programa.

## Ejercicio
Implementa una función que calcule la suma de los primeros 1000 números pares utilizando dos métodos diferentes: uno con un bucle `for` y otro con una comprensión de lista. Utiliza `timeit` para medir el tiempo de ejecución de cada método y compara los resultados.

## Resumen
- Se aprendió a utilizar herramientas como `timeit` y `cProfile` para medir y analizar el rendimiento del código.
- Se destacó la importancia de elegir las estructuras de datos adecuadas y optimizar bucles para mejorar la eficiencia.
