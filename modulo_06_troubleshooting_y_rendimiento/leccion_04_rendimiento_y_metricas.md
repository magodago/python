# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas en Python, se explorará cómo medir y optimizar la eficiencia de los programas. Se aprenderá a utilizar herramientas como `timeit` para medir el tiempo de ejecución y `cProfile` para analizar el rendimiento detallado del código. También se discutirá la importancia de seleccionar las estructuras de datos adecuadas y optimizar bucles, así como cómo implementar técnicas de programación funcional para mejorar la eficiencia.

## Ejercicio
Implementa un programa que calcule la suma de los primeros 100.000 números enteros utilizando diferentes métodos: una solución tradicional con un bucle `for`, y otra usando programación funcional con `sum()` y `range()`. Luego, utiliza `timeit` para medir el tiempo de ejecución de cada método y compara los resultados.

## Resumen
- Se aprendió a utilizar herramientas como `timeit` y `cProfile` para evaluar y optimizar el rendimiento del código.
- Se destacó la importancia de elegir estructuras de datos adecuadas y optimizar bucles.
- Se practicó la implementación de técnicas de programación funcional para mejorar la eficiencia.
