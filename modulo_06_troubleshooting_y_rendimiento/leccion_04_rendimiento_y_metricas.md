# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas de Python, se explorará cómo medir y optimizar la eficiencia del código. Se aprenderá a utilizar herramientas como `timeit` para medir el tiempo de ejecución y `cProfile` para analizar la eficiencia del rendimiento del programa. Además, se discutirá la importancia de elegir las estructuras de datos adecuadas y optimizar bucles para mejorar el rendimiento general.

## Ejercicio
Implementa un algoritmo que calcule los primeros 1000 números primos utilizando dos diferentes métodos: uno con una lista y otro con un generador. Utiliza `timeit` para medir el tiempo de ejecución de cada método e identifica cuál es más eficiente.

## Resumen
- Se aprendió a utilizar herramientas como `timeit` y `cProfile` para medir y analizar el rendimiento del código.
- Se discutió la importancia de elegir las estructuras de datos adecuadas y optimizar bucles para mejorar el rendimiento.
