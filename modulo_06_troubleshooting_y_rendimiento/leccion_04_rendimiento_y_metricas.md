# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas, se explorará cómo medir la eficiencia del código Python y cómo optimizar su ejecución. Se presentarán herramientas como `timeit`, `cProfile` y `line_profiler` para analizar el tiempo de ejecución y el perfil de rendimiento de las funciones. También se discutirá la importancia de seleccionar métricas apropiadas para evaluar el rendimiento de un programa, incluyendo tiempos de ejecución, velocidad de procesamiento y uso de memoria.

## Ejercicio
El estudiante deberá analizar un código Python que realiza operaciones matemáticas complejas. Utilizando las herramientas `timeit` y `cProfile`, deben medir el tiempo de ejecución y generar un perfil de rendimiento del programa. Posteriormente, se les pedirá que optimicen el código para mejorar su eficiencia y vuelvan a medir los tiempos y métricas.

## Resumen
- Se aprende a utilizar herramientas como `timeit` y `cProfile` para medir el rendimiento del código.
- Se entiende la importancia de seleccionar las métricas adecuadas para evaluar el rendimiento.
- Se practica la optimización de código mediante el análisis de métricas.
