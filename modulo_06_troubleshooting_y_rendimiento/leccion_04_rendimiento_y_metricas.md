# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas, se explorará cómo medir y optimizar la eficiencia de un código Python. Los estudiantes aprenderán a utilizar herramientas como `timeit` para medir el tiempo de ejecución y `cProfile` para analizar el rendimiento del programa en profundidad.

Se discutirá la importancia de comprender las métricas relevantes, tales como tiempos de ejecución, uso de memoria y frecuencia de llamadas a funciones. Se proporcionará un ejemplo práctico donde se compararán diferentes implementaciones de una función para determinar cuál es más eficiente.

## Ejercicio
Los estudiantes deberán escribir una función que calcule la suma de los primeros 1000 números enteros y luego medir su tiempo de ejecución utilizando `timeit`. A continuación, deben optimizar la función utilizando listas comprensión o otras técnicas y volver a medir el tiempo. Compararán ambos tiempos y analizarán las diferencias.

## Resumen
- Se aprenderá a utilizar herramientas como `timeit` y `cProfile` para medir y analizar el rendimiento de un código Python.
- Se entenderá la importancia de medir y comparar diferentes implementaciones para optimizar el rendimiento.
