# Rendimiento y metricas

## Contenido
En esta lección sobre rendimiento y métricas, se explorará cómo medir y optimizar la eficiencia del código Python. Se discutirá la importancia de utilizar herramientas de medición como `timeit` para evaluar el tiempo de ejecución y `cProfile` para analizar la utilización de recursos. También se presentarán conceptos clave como complejidad algorítmica y cómo seleccionar las estructuras de datos apropiadas pueden mejorar significativamente el rendimiento del código.

## Ejercicio
Implementa un programa que calcule los primeros 100 números de Fibonacci utilizando dos métodos: recursión y iteración. Utiliza `timeit` para medir la velocidad de ejecución de cada método y compara los resultados. Luego, utiliza `cProfile` para analizar las llamadas a funciones en ambos métodos y discute cuál es más eficiente en términos de rendimiento.

## Resumen
- Se aprendió cómo utilizar herramientas como `timeit` y `cProfile` para medir el tiempo de ejecución y analizar la utilización de recursos.
- Se destacó la importancia de considerar la complejidad algorítmica y la selección adecuada de estructuras de datos para mejorar el rendimiento del código.
