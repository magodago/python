Módulo 4: Tecnicas avanzadas

Proyecto practico 1

  • Implementar un sistema de gestión básica de inventario utilizando clases y objetos.
  • Utilizar métodos avanzados como herencia y polimorfismo para mejorar la estructura del código.

En esta lección, se profundizará en el uso de técnicas avanzadas de programación orientada a objetos con Python. Se trabajará en un proyecto práctico que consiste en desarrollar un sistema de gestión de inventario básico. Este sistema permitirá al usuario realizar operaciones como añadir productos, actualizar stock y buscar productos por nombre o categoría. El uso de herencia y polimorfismo facilitará la extensión del sistema para incluir diferentes tipos de productos con características específicas.

Para el ejercicio, los alumnos deberán crear una clase base `Producto` que contenga atributos como nombre, precio y cantidad en stock. Luego, se deben crear subclases como `Libro`, `Electrónica` y `Ropa`, cada una con sus propias características y métodos específicos. Finalmente, se implementará un menú interactivo para permitir al usuario realizar operaciones de gestión del inventario.

  • Se ha creado una estructura de clases que refleja la jerarquía de los productos.
  • El sistema permite la extensión fácil de nuevos tipos de productos a través de herencia.
