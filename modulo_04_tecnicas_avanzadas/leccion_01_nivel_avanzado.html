Módulo 4: Tecnicas avanzadas

Nivel avanzado

Objetivos

  • Comprender la optimización de código mediante la utilización de decoradores personalizados.
  • Aprender a implementar y utilizar generadores y corrutinas para mejorar el rendimiento y eficiencia del código.

Contenido

En esta lección avanzada, se explorará cómo mejorar significativamente el rendimiento y la legibilidad del código utilizando técnicas como los decoradores personalizados. Los decoradores son una característica poderosa de Python que permiten modificar o extender las funcionalidades de funciones y métodos sin alterar su estructura original.

Además, se profundizará en la utilización de generadores y corrutinas para manejar datos iterables de manera eficiente. Los generadores ofrecen una forma concisa y eficaz de crear iteradores que pueden producir secuencias de valores sobre la marcha, lo que reduce el uso de memoria y mejora el rendimiento en comparación con las listas o otros métodos tradicionales.

Ejercicio

Implemente un decorador personalizado llamado `tiempo_ejecucion` que muestre cuánto tiempo tarda una función en ejecutarse. Luego, utilice este decorador para medir el tiempo de ejecución de una función que calcule los primeros 100 números de Fibonacci utilizando un generador.

Resumen

  • Los decoradores personalizados son una herramienta poderosa para modificar funcionalidades de funciones y métodos.
  • Generadores y corrutinas permiten manejar datos iterables de manera eficiente, mejorando el rendimiento del código.
