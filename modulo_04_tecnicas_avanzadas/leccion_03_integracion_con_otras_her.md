# Integracion con otras herramientas

### Objetivos
- Comprender cómo Python puede integrarse con otras herramientas y bibliotecas para ampliar su funcionalidad.
- Aprender a utilizar interfaces de programación de aplicaciones (APIs) en Python.

### Contenido
La integración de Python con otras herramientas y bibliotecas es una técnica avanzada que permite aprovechar la potencia de diferentes sistemas y servicios. Por ejemplo, se puede integrar Python con bases de datos para realizar consultas complejas o automatizar tareas de análisis de datos. Además, la utilización de APIs (Interfaces de Programación de Aplicaciones) en Python facilita el acceso a servicios web como Google Maps API para geolocalización, o la API de Twitter para analizar tweets en tiempo real. Estas integraciones no solo mejoran la eficiencia del código sino que también permiten solucionar problemas más complejos y amplios.

### Ejercicio
Utilice la biblioteca `requests` en Python para hacer una solicitud GET a la API de OpenWeatherMap e imprimir la temperatura actual en Madrid, España. Primero, instale el paquete con el siguiente comando:
```bash
pip install requests
```
Luego, escriba un script que recupere y muestre la temperatura actual.

### Resumen
- Integración de Python con otras herramientas permite ampliar su funcionalidad.
- Uso de APIs en Python facilita el acceso a servicios web externos.
