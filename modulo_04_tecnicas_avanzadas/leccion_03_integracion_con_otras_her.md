# Integracion con otras herramientas

## Contenido
En esta lección del módulo de técnicas avanzadas sobre integración con otras herramientas, se explorará cómo Python puede ser utilizado como un puente entre diferentes sistemas y herramientas. Se abordarán conceptos como la integración con bases de datos mediante bibliotecas como SQLAlchemy o PyMySQL, así como el uso de APIs externas utilizando librerías como requests. Además, se presentarán ejemplos prácticos de cómo Python puede ser utilizado para automatizar tareas complejas que involucran múltiples herramientas y sistemas.

## Ejercicio
El estudiante deberá desarrollar un script en Python que realice la siguiente tarea: conectarse a una base de datos MySQL, ejecutar una consulta SQL para obtener información relevante, y luego enviar esa información a una API REST para procesarla. El ejercicio debe incluir el manejo adecuado de excepciones y la documentación clara del código.

## Resumen
- Se aprendió cómo integrar Python con bases de datos externas.
- Se exploró el uso de librerías como requests para interactuar con APIs externas.
- Se practicó la automatización de tareas que involucran múltiples herramientas y sistemas.
