# Integracion con otras herramientas

### Objetivos
- Comprender cómo integrar Python con bases de datos SQL.
- Aprender a utilizar bibliotecas externas como NumPy y Pandas para mejorar la funcionalidad del código.

### Contenido
En esta lección, profundizaremos en técnicas avanzadas de integración entre Python y otras herramientas. Comenzaremos explorando cómo Python puede interactuar con bases de datos SQL mediante el uso de bibliotecas como `sqlite3` y `psycopg2`. Aprenderemos a ejecutar consultas SQL desde Python, manejar resultados y realizar operaciones básicas de inserción, actualización y eliminación. Además, se abordarán aspectos importantes como la gestión de transacciones y la prevención de inyección SQL.

Posteriormente, se presentará el uso de bibliotecas externas para ampliar las capacidades del lenguaje. Se introducirá `NumPy` para realizar operaciones numéricas avanzadas y `Pandas` para manipulación de datos tabulares. Se explicarán conceptos clave como series y DataFrames, y se demostrarán ejemplos prácticos de cómo integrar estas bibliotecas con código Python existente.

### Ejercicio
Escribe un script en Python que:
1. Conecte a una base de datos SQLite.
2. Cree una tabla llamada `empleados` con las columnas `id`, `nombre` y `edad`.
3. Inserte tres registros nuevos en la tabla.
4. Consulte todos los registros de la tabla e imprima el resultado.

### Resumen
- Python puede integrarse eficazmente con bases de datos SQL mediante bibliotecas como `sqlite3` y `psycopg2`.
- Las bibliotecas externas `NumPy` y `Pandas` amplían significativamente las capacidades de Python para manipulación numérica y de datos tabulares.
