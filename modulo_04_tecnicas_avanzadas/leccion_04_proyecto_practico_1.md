# Proyecto practico 1

## Objetivos
- Implementar un proyecto práctico que integre técnicas avanzadas de Python.
- Desarrollar habilidades en la optimización y el rendimiento del código.

## Contenido
En esta lección, se enfocará en la aplicación práctica de las técnicas avanzadas de Python. Se trabajará con un proyecto que requiere la integración de múltiples conceptos, como programación orientada a objetos (POO), manejo de excepciones, y el uso de bibliotecas externas para mejorar el rendimiento del código. El objetivo es crear una aplicación simple que permita a los estudiantes aplicar sus conocimientos en un entorno real.

Se explorará cómo la POO puede ser utilizada para organizar y modularizar el código, mejorando así su legibilidad y mantenibilidad. Además, se abordará la importancia del manejo adecuado de excepciones para evitar errores y hacer que el programa sea más robusto. Finalmente, se discutirá cómo utilizar bibliotecas como NumPy o Pandas para optimizar operaciones complejas.

## Ejercicio
Desarrolla una aplicación simple en Python que permita al usuario ingresar datos de ventas diarias de un pequeño negocio. La aplicación debe:

1. Usar POO para crear una clase `Venta` con atributos como fecha, producto y monto.
2. Implementar métodos para agregar nuevas ventas, calcular el total mensual y generar informes resumidos.
3. Utilizar excepciones para manejar entradas incorrectas del usuario.
4. Optimiza la búsqueda de ventas específicas utilizando una biblioteca como Pandas.

## Resumen
- Integración de técnicas avanzadas en un proyecto práctico.
- Uso de POO y excepciones para mejorar el código.
- Optimización del rendimiento con la ayuda de bibliotecas externas.
