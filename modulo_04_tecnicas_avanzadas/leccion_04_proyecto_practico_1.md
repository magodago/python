# Proyecto practico 1

- Comprender la integración de bibliotecas externas como NumPy y Pandas para manipulación de datos.
- Desarrollar habilidades en la creación de visualizaciones con Matplotlib para análisis de datos.

Para este proyecto práctico, los estudiantes deberán trabajar con un conjunto de datos reales proporcionado por el profesor. Los pasos a seguir son:

1. Importar y explorar los datos utilizando NumPy y Pandas.
2. Realizar limpieza de datos necesaria.
3. Crear visualizaciones para analizar las relaciones entre variables en el dataset.

El objetivo es que los estudiantes adquieran experiencia práctica en la manipulación y análisis de datos utilizando herramientas populares del ecosistema Python, fortaleciendo sus habilidades en el manejo avanzado de datos.

- Integración efectiva de NumPy y Pandas para el procesamiento de datos.
- Creación de visualizaciones informativas con Matplotlib.
