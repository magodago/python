# Proyecto practico 1

## Objetivos
- Desarrollar un proyecto que permita a los alumnos aplicar técnicas avanzadas de programación en Python.
- Fomentar la resolución de problemas complejos mediante la implementación de algoritmos eficientes y estructuras de datos.

## Contenido
En esta lección, se continuará con el desarrollo del Proyecto Práctico 1, enfocándose en técnicas avanzadas. Los alumnos deberán aplicar conceptos como programación orientada a objetos (POO), manejo de excepciones, y la optimización de código para resolver un problema real. El proyecto consiste en crear una aplicación que permita gestionar una biblioteca digital, incluyendo funcionalidades como el registro de libros, gestión de préstamos y devoluciones.

Se introducirá la utilización de clases y objetos para representar diferentes entidades del sistema (como libros, usuarios y transacciones), así como métodos para realizar operaciones sobre estos objetos. Se abordará también cómo manejar excepciones para mejorar la robustez del programa y proporcionar un feedback adecuado al usuario.

## Ejercicio
Desarrolla una clase `Libro` con los siguientes atributos: título, autor, ISBN, año de publicación y estado (disponible o prestado). Implementa métodos para modificar el estado del libro entre disponible y prestado. Crea un menú interactivo donde se puedan registrar nuevos libros, mostrar la lista de libros disponibles, realizar préstamos y devoluciones.

## Resumen
- Se enfatiza en la aplicación práctica de técnicas avanzadas como POO.
- Los alumnos implementarán funcionalidades complejas en una aplicación real.
- El ejercicio fomenta el uso de excepciones y la gestión de estados en objetos.
