# Proyecto practico 1

- Dominar la integración de módulos personalizados y estándar para resolver problemas complejos.
- Implementar técnicas avanzadas de programación orientada a objetos para mejorar la legibilidad y mantenibilidad del código.

La lección se centra en aplicar técnicas avanzadas de Python, enfocándose en el desarrollo de un proyecto práctico que integre múltiples módulos. Los estudiantes aprenderán cómo combinar funcionalidades de diferentes módulos para crear soluciones robustas y eficientes. Se explorará la importancia de modularizar el código y la implementación de patrones de diseño para optimizar su estructura.

Para este ejercicio, los alumnos deberán desarrollar un sistema simple que administre una biblioteca virtual. Deberán utilizar módulos personalizados para gestionar libros, autores y prestamos, así como módulos estándar como `datetime` y `csv` para manejar fechas y datos de archivo.

- Modularización del código
- Uso eficiente de módulos estándar y personalizados
