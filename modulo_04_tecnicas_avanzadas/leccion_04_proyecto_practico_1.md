# Proyecto practico 1

## Objetivos
- Implementar un proyecto práctico que utilice técnicas avanzadas de Python para mejorar la eficiencia y funcionalidad del código.
- Aprender a utilizar módulos y bibliotecas externas para resolver problemas complejos.

## Contenido
En esta lección, se profundizará en el uso de módulos y bibliotecas externas en Python. Se explorará cómo importar y utilizar módulos predefinidos como `os`, `sys` y `datetime`. También se presentará la instalación y utilización de paquetes con pip, así como la integración de bibliotecas populares como NumPy para operaciones matemáticas avanzadas y Pandas para el análisis de datos. Se enfatizará la importancia de elegir las herramientas adecuadas según las necesidades del proyecto.

## Ejercicio
Desarrolla un script que use la biblioteca `Pandas` para leer un archivo CSV, realizar operaciones básicas de análisis de datos (como contar filas y columnas, calcular promedios) y guardar los resultados en un nuevo archivo CSV. Utiliza el módulo `os` para manejar rutas del sistema y asegúrate de que tu script sea portable.

## Resumen
- Se aprendió a importar y utilizar módulos predefinidos.
- Se practicó la instalación e integración de paquetes externos con pip.
- Se aplicaron técnicas avanzadas para mejorar el análisis de datos.
