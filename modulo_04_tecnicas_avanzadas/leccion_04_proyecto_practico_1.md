# Proyecto practico 1

## Objetivos
- Implementar un sistema de gestión de inventario utilizando clases y objetos.
- Utilizar herencia para modelar relaciones entre diferentes tipos de productos.

## Contenido
En esta lección avanzada de Python, se profundizará en la creación de un proyecto práctico que consiste en desarrollar un sistema de gestión de inventario. El objetivo es aplicar técnicas avanzadas como la programación orientada a objetos (POO) y el uso de herencia para crear una estructura modular y eficiente. Se comenzará definiendo una clase base `Producto` con atributos comunes, seguida de clases derivadas que representen diferentes tipos de productos específicos, como libros, electrodomésticos o alimentos. Cada subclase tendrá sus propios atributos y métodos únicos.

## Ejercicio
Implementa un sistema de gestión de inventario donde:
1. Define una clase base `Producto` con los atributos `nombre`, `codigo` y `precio`.
2. Crea clases derivadas como `Libro`, `Electrodomestico` y `Alimento`, cada una con atributos específicos (por ejemplo, `autor` para Libro, `potencia` para Electrodomestico).
3. Implementa métodos en las subclases que reflejen su funcionalidad específica.
4. Crea un inventario inicializado con al menos 5 productos de diferentes tipos y muestra cómo se pueden agregar nuevos productos o actualizar el precio existente.

## Resumen
- Se ha creado una estructura de clases utilizando POO para gestionar diferentes tipos de productos.
- La herencia se utilizó para definir subclases que heredan atributos comunes pero también tienen características únicas.
