# Proyecto practico 1

- Comprender la integración de técnicas avanzadas de Python para resolver problemas complejos.
- Implementar patrones de diseño y bibliotecas de terceros para mejorar la eficiencia del código.

En esta lección, se explorará el desarrollo de un proyecto práctico que combina varias técnicas avanzadas de Python. El objetivo es construir una aplicación web simple utilizando Flask como framework y SQLite como base de datos. Se utilizarán patrones de diseño como Singleton para gestionar la conexión a la base de datos y Decoradores para manejar autenticación de usuarios, lo que permitirá al alumno aplicar conocimientos teóricos en un entorno real.

Para el ejercicio, los estudiantes deberán desarrollar una aplicación web que permita a los usuarios registrar y acceder a cuentas. La aplicación debe incluir funcionalidades básicas como la creación de usuarios, inicio de sesión y cierre de sesión. Además, se deben implementar decoradores para proteger las rutas sensibles y asegurar que solo usuarios autenticados puedan acceder.

Los puntos clave del ejercicio son:
- Integración efectiva de Flask y SQLite.
- Uso de patrones de diseño Singleton y Decoradores en el código.
- Desarrollo de una aplicación web funcional con autenticación básica.
