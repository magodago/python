# Nivel avanzado

### Objetivos
- Comprender y aplicar técnicas avanzadas de programación con Python para optimizar el rendimiento del código.
- Implementar la programación orientada a objetos de manera más eficiente y robusta.

### Contenido
En esta lección, profundizaremos en las técnicas avanzadas de Python, enfocándonos en cómo mejorar la eficiencia y mantenibilidad del código. Veremos la optimización de funciones y clases, el uso de decoradores para modificar comportamientos de manera elegante, y la implementación de patrones de diseño para resolver problemas comunes de forma más estructurada.

### Ejercicio
Escribe una función que use un decorador para medir el tiempo de ejecución de otra función. Luego, aplica este decorador a una función que realice operaciones complejas con listas grandes y compara su rendimiento antes y después del uso del decorador.

### Resumen
- Se aprendió sobre la optimización de funciones y clases.
- Se introdujo el concepto de decoradores para mejorar la funcionalidad del código.
- Se destacó la importancia de patrones de diseño en la resolución de problemas complejos.
