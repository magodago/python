# Revision

## Objetivos
- Consolidar el conocimiento sobre las técnicas avanzadas de programación en Python.
- Aprender a utilizar eficazmente las bibliotecas y módulos de terceros.

## Contenido
En esta lección, revisaremos técnicas avanzadas de programación con Python, enfocándonos en la optimización del código y el uso efectivo de bibliotecas externas. Estudiarás cómo mejorar el rendimiento del código a través de la utilización de decoradores, comprensiones de listas y generadores. Además, profundizarás en la importancia y el uso de paquetes como NumPy y Pandas para manejo eficiente de datos.

## Ejercicio
Implementa una función que use un decorador para medir el tiempo de ejecución de otra función. Luego, compara los tiempos de ejecución entre dos implementaciones: una sin optimización y otra utilizando generadores. Utiliza la biblioteca `time` para medir el tiempo.

## Resumen
- Se revisaron técnicas avanzadas como decoradores y comprensiones de listas.
- Se aprendió a utilizar eficazmente bibliotecas externas como NumPy y Pandas.
- Se practicó la optimización del código mediante el uso de generadores.
