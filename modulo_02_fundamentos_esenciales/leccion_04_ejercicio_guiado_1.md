# Ejercicio guiado 1

- Familiarizarse con la estructura básica de un script Python.
- Practicar la creación y uso de variables para almacenar datos.

En esta lección, aprenderás a escribir tu primer programa en Python. Comenzaremos por entender cómo estructurar un script básico, incluyendo la indentación que es crucial en Python para indicar bloques de código. A continuación, te guiaré paso a paso para crear variables y asignarles diferentes tipos de datos como cadenas de texto, números enteros y flotantes.

Ejercicio: Escribe un programa simple que solicite al usuario su nombre e imprima una saludo personalizado. Primero, declara una variable `nombre` y asigna a esta el valor ingresado por el usuario. Luego, utiliza la función `print()` para mostrar en pantalla "¡Hola, [nombre]!".

Resumen:
- Importante entender la importancia de la indentación en Python.
- Práctica en la creación y uso de variables para almacenar datos de diferentes tipos.
