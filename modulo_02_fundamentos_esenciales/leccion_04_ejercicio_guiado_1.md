# Ejercicio guiado 1

Objetivos
- Comprender la sintaxis básica de Python.
- Practicar la creación y uso de variables.

Contenido
En esta lección del módulo Fundamentos esenciales, se explorará el inicio del viaje hacia el dominio de Python. Se iniciará con una introducción a la sintaxis básica del lenguaje, enfocándose en cómo escribir y ejecutar un programa simple. Los estudiantes aprenderán sobre las variables, sus tipos básicos (enteros, flotantes, cadenas) y cómo declararlas. Además, se practicará la asignación de valores y el uso de estas variables en operaciones simples.

Ejercicio
Crea un programa que realice lo siguiente:
1. Declara tres variables: una entera llamada `edad`, una flotante llamada `altura` y una cadena llamada `nombre`.
2. Asigna los valores 30, 5.9 y "Juan" a estas variables respectivamente.
3. Imprime en pantalla el mensaje: "Mi nombre es [nombre], tengo [edad] años y mido [altura] metros."

Resumen
- Se aprendió la sintaxis básica de Python para declarar y asignar valores a variables.
- Se practicó la creación de un programa simple que utiliza diferentes tipos de datos.
