# Ejercicio guiado 1

### Objetivos
- Familiarizar al estudiante con la sintaxis básica de Python.
- Introducir los conceptos fundamentales de las variables y tipos de datos.

### Contenido
En esta lección, exploraremos los primeros pasos en el mundo del lenguaje de programación Python. Comenzaremos con una breve introducción a la sintaxis básica, incluyendo cómo escribir y ejecutar un programa simple. A continuación, profundizaremos en las variables y tipos de datos más comunes utilizados en Python, como enteros, flotantes, cadenas de texto y booleanos. Estos conceptos son esenciales para cualquier programador que desee dominar Python.

Para ilustrar estos conceptos, veremos ejemplos prácticos donde se crean variables con diferentes tipos de datos y se realizan operaciones básicas con ellos. A través de este ejercicio guiado, los estudiantes aprenderán a escribir código eficiente y comprensible en Python.

### Ejercicio
Escribe un programa en Python que realice lo siguiente:
1. Declara una variable `edad` y asigna un valor entero.
2. Declara una variable `altura` y asigna un valor de punto flotante.
3. Declara una variable `nombre` y asigna un valor de cadena de texto.
4. Declara una variable `es_estudiante` y asigna un valor booleano.

Luego, imprime el tipo de cada variable utilizando la función `type()` incorporada en Python.

### Resumen
- Familiarización con la sintaxis básica de Python.
- Compreención de variables y tipos de datos fundamentales.
