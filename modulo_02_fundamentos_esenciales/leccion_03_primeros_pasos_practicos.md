# Primeros pasos practicos

Objetivos
- Familiarizar al estudiante con la instalación y configuración de Python.
- Introducir a los estudiantes en la sintaxis básica de Python.

Contenido
En esta lección, comenzaremos explorando cómo instalar Python en su sistema operativo. Asegúrese de elegir una versión compatible (3.x) para obtener las características más recientes del lenguaje. Posteriormente, introduciremos los conceptos básicos de la sintaxis de Python, incluyendo la estructura de un programa simple y cómo escribir su primera línea de código.

Ejercicio
Escriba un pequeño programa que imprima "¡Hola, mundo!" en la consola. Guarde el archivo con el nombre `hola_mundo.py`. Luego, ejecute el programa desde la terminal o línea de comandos usando Python 3 y verifique que se muestre el mensaje esperado.

Resumen
- Se ha instalado correctamente Python.
- Se ha escrito y ejecutado un programa simple en Python.
