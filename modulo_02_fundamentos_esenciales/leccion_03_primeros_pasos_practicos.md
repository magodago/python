# Primeros pasos practicos

- Comprender la estructura básica de un programa en Python.
- Familiarizarse con la sintaxis y los comentarios.

En esta lección, aprenderás a escribir tu primer programa en Python y entenderás cómo organizarlo correctamente. Veremos cómo iniciar un archivo de Python, escribir código simple y agregar comentarios para hacer el código más legible. La estructura básica incluye declaraciones de importación (si es necesario), la definición de funciones o bloques de código, y finalmente la ejecución del programa.

Para practicar estos conceptos, crea un archivo llamado `hola_mundo.py` en tu editor de texto favorito. Escribe el siguiente código:

```python
# Este es un comentario que explica lo que hace el programa

print("¡Hola, mundo!")
```

Guarda el archivo y ejecútalo en la terminal o en un intérprete interactivo de Python para ver los resultados.

- Comenzar a escribir programas simples.
- Utilizar comentarios para documentar código.
