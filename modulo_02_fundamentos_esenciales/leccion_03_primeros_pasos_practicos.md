# Primeros pasos practicos

Objetivos
- Familiarizar a los estudiantes con la instalación y configuración de Python.
- Introducir conceptos básicos de sintaxis y ejecución de código.

Contenido
En esta lección, los estudiantes aprenderán cómo instalar Python en su sistema operativo preferido y cómo verificar si la instalación fue exitosa. Se introducirá la estructura básica de un programa en Python, incluyendo las convenciones de sintaxis como indentación y el uso de comillas para cadenas de texto. Los estudiantes también aprenderán a ejecutar sus primeros programas desde la línea de comandos y cómo utilizar herramientas de editor de código simple.

Ejercicio
Escriba un programa en Python que imprima "¡Hola, mundo!" en la consola. Guarde el archivo con el nombre `hola_mundo.py` y ejecute el script desde la terminal o línea de comandos. Asegúrese de verificar que la salida sea exactamente como se espera.

Resumen
- Se ha instalado Python correctamente.
- Se ha escrito, ejecutado y verificado un programa simple en Python.
