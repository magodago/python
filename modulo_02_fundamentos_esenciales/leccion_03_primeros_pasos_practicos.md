# Primeros pasos practicos

Objetivos
- Familiarizar a los estudiantes con la instalación y configuración de Python.
- Introducir conceptos básicos como las variables y tipos de datos primitivos.

Contenido
En esta lección, exploraremos cómo instalar y configurar Python en nuestro entorno de trabajo. Es crucial que cada estudiante tenga Python correctamente instalado para poder seguir los ejercicios prácticos. A continuación, aprenderemos a asignar valores a variables y trabajar con diferentes tipos de datos primitivos como enteros, flotantes y cadenas.

Para comenzar, es importante saber que existen varias versiones de Python (Python 2.x y Python 3.x), pero se recomienda utilizar la versión 3.x para aprovechar las mejoras y nuevas características. Podemos descargar Python desde el sitio oficial de Python (https://www.python.org/downloads/). Luego, siguiendo las instrucciones en la página de descargas, instalaremos Python en nuestro sistema.

Ejercicio
Práctica: Instalar Python y crear un programa simple

1. Descarga e instala Python 3.x desde el sitio oficial.
2. Abre una terminal o línea de comandos y verifica que Python esté correctamente instalado escribiendo `python --version` (o `python3 --version` en sistemas Unix).
3. Crea un archivo llamado `primer_programa.py`.
4. Escriba el siguiente código dentro del archivo:
   ```python
   nombre = "Juan"
   edad = 25
   print(f"Hola, me llamo {nombre} y tengo {edad} años.")
   ```
5. Ejecuta el programa en la terminal escribiendo `python primer_programa.py` o `python3 primer_programa.py`.

Resumen
- Se ha instalado Python correctamente.
- Se han creado variables y se han utilizado tipos de datos primitivos para imprimir un mensaje personalizado.
