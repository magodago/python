# Terminologia basica

- Comprender los conceptos fundamentales de programación relacionados con Python.
- Familiarizarse con la terminología básica utilizada en el desarrollo de programas con Python.

Python es un lenguaje de programación de alto nivel que se utiliza ampliamente para desarrollar aplicaciones de todo tipo, desde sitios web hasta software empresarial. En este módulo, se introducirá a los estudiantes a algunos términos clave que son esenciales para comprender cómo funciona Python y cómo estructurar programas en él. Estos incluyen conceptos como variables, tipos de datos, funciones y control de flujo.

Las variables en Python son contenedores que almacenan valores. Pueden ser de diferentes tipos, como enteros, cadenas o listas. Las funciones son bloques de código reutilizables que realizan una tarea específica y pueden recibir parámetros para personalizar su comportamiento. El control de flujo se refiere a la estructura del programa, especificando qué código se ejecuta en qué momento.

Para practicar, los estudiantes deberán escribir un programa simple que solicite al usuario su nombre y edad, luego muestre estos datos en pantalla. Deberán utilizar variables para almacenar el nombre y la edad ingresados por el usuario, y una función para imprimirlos.

- Control de flujo.
