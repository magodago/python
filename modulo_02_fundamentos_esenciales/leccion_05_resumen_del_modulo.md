# Resumen del modulo

Objetivos
- Consolidar la comprensión de las estructuras básicas del lenguaje Python.
- Aprender a utilizar los tipos de datos fundamentales y operaciones con ellos.

Contenido
En esta lección, se revisará el contenido del módulo "Fundamentos esenciales" para asegurar que los estudiantes tengan una sólida base en Python. Se enfocará en las estructuras básicas del lenguaje, incluyendo la sintaxis y gramática fundamental. Los tipos de datos más comunes en Python, como enteros, flotantes, cadenas de texto y listas, serán examinados junto con operaciones básicas que se pueden realizar sobre ellos. Se abordarán también conceptos avanzados como las estructuras condicionales y bucles.

Ejercicio
Implemente un programa en Python que solicite al usuario ingresar su nombre y edad. El programa debe mostrar un mensaje que indique cuántos años tendrá el usuario en 10 años. Utilice variables para almacenar los datos ingresados y operaciones matemáticas básicas para calcular la edad futura.

Resumen
- Se revisaron las estructuras básicas del lenguaje Python.
- Los tipos de datos fundamentales y sus operaciones fueron estudiados.
- Se practicó el uso de variables, condicionales y bucles.
