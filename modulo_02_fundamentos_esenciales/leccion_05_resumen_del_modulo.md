# Resumen del modulo

- Comprender los fundamentos esenciales de Python.
- Aplicar conocimientos básicos para escribir y ejecutar programas simples.

Python es un lenguaje de programación de alto nivel que se destaca por su simplicidad y claridad. Este módulo ha proporcionado una introducción a las estructuras básicas del lenguaje, incluyendo variables, tipos de datos, operadores aritméticos y lógicos, así como la sintaxis para definir funciones y controlar flujos de ejecución con estructuras condicionales y bucles. Estas herramientas forman el núcleo de cualquier programa en Python.

Para practicar estos conceptos, los estudiantes deben desarrollar un pequeño programa que calcule el área de diferentes figuras geométricas (círculo, rectángulo y triángulo) utilizando fórmulas matemáticas. El programa debe permitir al usuario elegir la figura y proporcionar sus dimensiones para calcular e imprimir el área correspondiente.

- Dominio de las variables y tipos de datos en Python.
- Uso de estructuras condicionales y bucles para controlar el flujo del programa.
