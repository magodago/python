# Resumen del modulo

Objetivos
- Consolidar la comprensión de los conceptos fundamentales de Python.
- Aplicar conocimientos adquiridos sobre estructuras de datos y control de flujo.

Contenido
En esta lección, se revisará el resumen del módulo "Fundamentos esenciales" para asegurar que los estudiantes hayan asimilado los conceptos clave. Se abordaron temas como las variables y tipos de datos en Python, estructuras condicionales y bucles, así como la utilización de listas y diccionarios. Los estudiantes aprendieron a manipular estos elementos básicos para resolver problemas simples.

Se destacó la importancia de la sintaxis correcta y el uso adecuado de los operadores y funciones incorporadas en Python. Además, se exploraron ejemplos prácticos que ilustran cómo combinar estas herramientas para crear programas sencillos pero efectivos.

Ejercicio
Cree un programa en Python que solicite al usuario su nombre y edad. Utilice estructuras condicionales para determinar si la persona es mayor de 18 años. Si lo es, imprima "Eres mayor de edad". De lo contrario, imprima "Aún eres menor de edad". Asegúrese de manejar el caso en que el usuario ingrese una edad no válida.

Resumen
- Los estudiantes deben tener un sólido entendimiento de variables, tipos de datos y estructuras condicionales.
- La capacidad para utilizar listas y diccionarios eficazmente es crucial para resolver problemas prácticos.
