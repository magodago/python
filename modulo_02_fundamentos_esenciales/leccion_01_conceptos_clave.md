# Conceptos clave

## Objetivos
- Comprender los conceptos básicos de Python, incluyendo sintaxis y tipos de datos.
- Familiarizarse con las estructuras de control como if, else, y elif.

## Contenido
En esta lección del módulo "Fundamentos esenciales", se explorarán los conceptos clave que son fundamentales para cualquier programador iniciante en Python. Comenzaremos por entender la sintaxis básica, que incluye cómo escribir y ejecutar código de manera correcta. Luego, abordaremos los tipos de datos más comunes en Python, como enteros, flotantes, cadenas y booleanos. Finalmente, se presentarán las estructuras de control, especialmente el uso del condicional `if`, que permite la toma de decisiones en un programa.

## Ejercicio
Escribe un pequeño script en Python que solicite al usuario su edad y determine si es mayor de 18 años o no. Utiliza una estructura condicional para mostrar un mensaje diferente dependiendo de la respuesta.

```python
edad = int(input("¿Cuál es tu edad? "))
if edad >= 18:
    print("Eres mayor de 18 años.")
else:
    print("Eres menor de 18 años.")
```

## Resumen
- Se aprendió sobre la sintaxis básica y los tipos de datos en Python.
- Se introdujo el uso del condicional `if` para realizar decisiones en el código.
