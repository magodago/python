# Conceptos clave

## Objetivos
- Comprender los conceptos básicos de Python que son esenciales para cualquier programador.
- Aprender a utilizar variables y tipos de datos primitivos.

## Contenido
En esta lección, se introducirán los fundamentos esenciales del lenguaje de programación Python. Se comenzará por explicar qué es un entorno de desarrollo integrado (IDE) y cómo instalar Python en el ordenador para poder empezar a codificar. Posteriormente, se presentarán las variables y tipos de datos primitivos más utilizados, como enteros, flotantes, cadenas de texto y booleanos. Se ilustrará la importancia de asignar valores a estas variables y cómo manipularlos mediante operaciones básicas.

## Ejercicio
Escribe un programa simple en Python que solicite al usuario su nombre y edad, luego imprima un mensaje personalizado. El programa debe utilizar variables para almacenar el nombre y la edad del usuario, y realizar una operación matemática básica (por ejemplo, sumar 10 años a la edad actual).

```python
nombre = input("¿Cuál es tu nombre? ")
edad = int(input("¿Cuántos años tienes? "))
edad_futura = edad + 10

print(f"Hola {nombre}, en 10 años tendrás {edad_futura} años.")
```

## Resumen
- Python es un lenguaje de programación interpretado, de sintaxis clara y fácil de leer.
- Las variables son contenedores que almacenan datos, y los tipos de datos primitivos incluyen enteros, flotantes, cadenas de texto y booleanos.
