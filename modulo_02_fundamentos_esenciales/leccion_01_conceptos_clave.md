# Conceptos clave

## Objetivos
- Comprender los conceptos fundamentales de Python, incluyendo su sintaxis básica y tipos de datos.
- Aprender a escribir y ejecutar programas simples en Python.

## Contenido
En esta lección sobre los conceptos clave de Python, se introducirá la sintaxis básica del lenguaje. Los estudiantes aprenderán acerca de los tipos de datos primitivos como enteros, flotantes, cadenas y booleanos. Se explicará cómo declarar variables y asignarles valores, así como las operaciones básicas que se pueden realizar con estos tipos de datos. Además, se presentarán comentarios en el código para mejorar la legibilidad.

## Ejercicio
Escribe un programa simple en Python que haga lo siguiente:
1. Declara tres variables: una entera llamada `edad`, una flotante llamada `altura` y una cadena llamada `nombre`.
2. Asigna valores a estas variables.
3. Imprime en pantalla el mensaje "Mi nombre es [nombre], tengo [edad] años y mido [altura] metros".

## Resumen
- Python utiliza sintaxis clara y legible, con énfasis en la claridad del código.
- Los tipos de datos primitivos son fundamentales para comenzar a programar en Python.
