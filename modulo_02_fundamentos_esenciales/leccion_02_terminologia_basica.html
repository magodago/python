Módulo 2: Fundamentos esenciales

Terminologia basica

Objetivos

  • Comprender los conceptos básicos de programación relevantes para Python.
  • Familiarizarse con la terminología fundamental del lenguaje.

Contenido

En esta lección, se explorarán los conceptos esenciales y la terminología básica que son fundamentales para entender el funcionamiento de Python. Se abordarán temas como variables, tipos de datos, estructuras de control (condicionales y bucles) y funciones. Estas herramientas forman la base sobre la cual se construirá el conocimiento del estudiante en programación con Python.

Ejercicio

Implementa un pequeño programa que solicite al usuario su nombre y edad, muestre un mensaje personalizado y calcule cuántos años tendrá el usuario en 10 años. Utiliza variables para almacenar la información ingresada por el usuario, tipos de datos apropiados para la edad (entero) y el mensaje (cadena), y una función para calcular la edad futura.

Resumen

  • Variables: elementos utilizados para almacenar datos.
  • Tipos de datos básicos: enteros, cadenas, booleanos.
  • Estructuras de control condicionales y bucles para tomar decisiones y repetir acciones.
