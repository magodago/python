Módulo 2: Fundamentos esenciales

Ejercicio guiado 1

Objetivos

  • Comprender la sintaxis básica de Python para la definición y uso de variables.
  • Aprender a imprimir texto en pantalla utilizando la función `print()`.

    • Contenido

    En esta lección, se explorará el concepto fundamental de las variables en Python. Las variables son contenedores que almacenan datos y son esenciales para realizar cálculos y manipular información dentro del programa. Se aprenderá a declarar variables con diferentes tipos de datos como cadenas de texto, números enteros y flotantes. Además, se introducirá la función `print()`, que permite mostrar texto o valores en la consola.

    Se explicará cómo asignar un valor a una variable usando el operador igual (`=`) y cómo utilizarla posteriormente para realizar operaciones simples o imprimir su contenido en pantalla. Se proporcionarán ejemplos prácticos para ilustrar cada concepto, desde la declaración de variables hasta la ejecución de comandos `print()`.

    Ejercicio

    Escribe un programa que declare tres variables: una llamada `nombre` con tu nombre como valor, otra llamada `edad` con tu edad como valor y finalmente una variable `ciudad` con el nombre de tu ciudad. Luego, imprime en la consola un mensaje que diga "Hola, me llamo [nombre], tengo [edad] años y vivo en [ciudad]". Reemplaza los corchetes con las variables correspondientes.

    Resumen

  • Se aprendió a declarar variables en Python para almacenar diferentes tipos de datos.
  • Se utilizó la función `print()` para mostrar texto o valores en la consola.
