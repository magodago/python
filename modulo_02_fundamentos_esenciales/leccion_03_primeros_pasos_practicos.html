Módulo 2: Fundamentos esenciales

Primeros pasos practicos

  • Comprender la estructura básica de un programa en Python.
  • Practicar la escritura y ejecución de programas simples.

Para comenzar, vamos a explorar cómo se escribe y ejecuta un programa simple en Python. Primero, debes tener instalado Python en tu computadora. Puedes verificar si ya lo tienes instalado abriendo una terminal o línea de comandos e ingresando `python --version` (o `python3 --version`, dependiendo de tu sistema operativo).

Un programa simple puede ser un script que imprime "¡Hola, mundo!" en la consola. Para escribir este programa, sigue estos pasos:

1. Abre un editor de texto como Notepad++, Sublime Text o Visual Studio Code.

2. Escribe el siguiente código:

```python

print("¡Hola, mundo!")

```

3. Guarda el archivo con extensión `.py`, por ejemplo, `hola_mundo.py`.

4. Abre una terminal y navega hasta la carpeta donde guardaste tu archivo.

5. Ejecuta el programa escribiendo `python hola_mundo.py` (o `python3 hola_mundo.py`, dependiendo de tu configuración) y presiona Enter.

Verás que se muestra "¡Hola, mundo!" en la consola.

### Resumen

  • Se aprendió a escribir un programa simple en Python.
  • Practicamos la ejecución de este programa desde una terminal.
