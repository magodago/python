Módulo 2: Fundamentos esenciales

Primeros pasos practicos

  • Familiarizar al estudiante con la instalación y configuración de Python.
  • Introducir a los estudiantes en la escritura básica de código Python.

Para comenzar, es importante que todos tengan Python instalado en su computadora. Pueden hacerlo desde el sitio web oficial de Python (python.org) siguiendo las instrucciones para su sistema operativo. Una vez instalado, pueden verificar la versión de Python ejecutando `python --version` en la terminal o comandos.

Ahora que tenemos Python configurado, podremos empezar a escribir nuestro primer programa: "Hola, Mundo!". Este es un clásico introducido en muchos lenguajes de programación para familiarizarse con el funcionamiento básico. Para ello, abrimos una nueva ventana del editor de texto (como Notepad++, Sublime Text o cualquier otro) y escribimos la siguiente línea:

```python

print("Hola, Mundo!")

```

Guardamos el archivo como `hola_mundo.py` en un directorio de nuestro elección. Para ejecutar este programa, abrimos una terminal o comandos, nos movemos al directorio donde guardamos el archivo y escribimos `python3 hola_mundo.py`. Si todo está configurado correctamente, deberíamos ver la salida "Hola, Mundo!" en la consola.

Este ejercicio nos permite familiarizarnos con el entorno de desarrollo y la sintaxis básica de Python. Es importante practicar este paso para asegurarse de que el entorno esté correctamente configurado antes de avanzar a otros conceptos más complejos.

  • Instalación y configuración de Python.
  • Escritura y ejecución del primer programa: "Hola, Mundo!".
