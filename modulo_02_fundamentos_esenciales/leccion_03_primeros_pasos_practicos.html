Módulo 2: Fundamentos esenciales

Primeros pasos practicos

  • Familiarizarse con la instalación y configuración de Python.
  • Escribir y ejecutar su primera línea de código.

Python es un lenguaje de programación de alto nivel que se utiliza ampliamente en desarrollo web, análisis de datos, inteligencia artificial e ingeniería. Para comenzar a utilizar Python, primero debes instalarlo en tu sistema operativo. Puedes descargar la última versión de Python desde el sitio oficial (https://www.python.org/downloads/). Durante la instalación, asegúrate de marcar la opción que agrega Python al PATH del sistema para facilitar su uso.

Para practicar los primeros pasos, abre una terminal o un editor de texto y ejecuta el siguiente código:

```python

print("¡Hola, mundo!")

```

Guarda este código en un archivo llamado `hola_mundo.py` y luego ejecútalo desde la terminal con el comando `python3 hola_mundo.py`. Deberías ver la salida "¡Hola, mundo!" en tu terminal.
