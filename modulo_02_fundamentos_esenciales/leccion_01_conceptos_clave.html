Módulo 2: Fundamentos esenciales

Conceptos clave

Objetivos

  • Comprender los conceptos fundamentales de Python.
  • Familiarizarse con la sintaxis básica y el entorno de desarrollo.

    • Contenido

    En esta lección se abordarán los conceptos clave que forman parte esencial del lenguaje de programación Python. Se comenzará por presentar la sintaxis básica, incluyendo cómo escribir y ejecutar un programa simple en Python. Luego, se explicarán los tipos de datos básicos como cadenas, enteros, flotantes y booleanos, así como las operaciones que se pueden realizar con ellos. Se ilustrará el uso de variables para almacenar y manipular información.

    Ejercicio

    Escribe un programa en Python que solicite al usuario su nombre, edad y altura. Luego, imprime una frase personalizada que incluya estos datos. Por ejemplo: "Hola [nombre], tienes [edad] años y mides [altura] metros."

    Resumen

  • Se aprende la sintaxis básica de Python.
  • Se familiariza con los tipos de datos fundamentales.
  • Se practica el uso de variables para almacenar información del usuario.
