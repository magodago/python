# Recursos adicionales

- Familiarizarse con recursos de documentación y tutoriales en línea.
- Identificar herramientas útiles para la práctica y mejora del código Python.

La lección sobre recursos adicionales se enfoca en proporcionar a los estudiantes una amplia gama de herramientas que pueden utilizar para aprender y mejorar su habilidad con el lenguaje de programación Python. Se recomienda la utilización de plataformas como Codecademy, Coursera y edX, que ofrecen cursos interactivos y gratuitos o de pago. Además, se sugiere familiarizarse con la documentación oficial de Python, disponible en python.org, así como con bibliotecas populares como NumPy y Pandas para trabajos de ciencia de datos.

Para el ejercicio práctico, los estudiantes deben elegir un recurso recomendado y realizar una pequeña introducción o resumen del mismo. Luego, deben explorar el recurso seleccionado durante al menos una hora, experimentando con ejemplos de código proporcionados e implementándolos en proyectos personales.

- Familiarizarse con diversas plataformas educativas para Python.
- Practicar y aplicar conocimientos a través del uso de recursos recomendados.
