# Requisitos previos

- Comprender los requisitos previos necesarios para comenzar a aprender Python.
- Identificar las habilidades y conocimientos básicos que facilitarán la comprensión del lenguaje de programación.

Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Para comenzar a aprender Python, se recomienda tener ciertos requisitos previos. Primero, es útil contar con una comprensión básica de conceptos matemáticos, especialmente en áreas como álgebra y lógica, ya que estos son fundamentales para la programación. Segundo, aunque no es absolutamente necesario, tener experiencia previa con otros lenguajes de programación puede facilitar el aprendizaje, ya que permitirá una mejor comprensión de los conceptos de estructura de datos y control de flujo. Finalmente, un entorno de desarrollo integrado (IDE) o un editor de texto como Sublime Text o Visual Studio Code es necesario para escribir y ejecutar código Python.

Ejercicio: Instale el software de Python en su computadora y utilice un IDE o editor de texto para escribir e imprimir la frase "¡Hola, mundo!" en pantalla. Asegúrese de que el resultado sea exactamente esta frase.
