# Requisitos previos

- Comprender los requisitos previos necesarios para comenzar a aprender Python.
- Identificar las habilidades básicas y conocimientos que facilitarán la adquisición de nuevas competencias en programación con Python.

Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Su sintaxis clara y concisa hace que sea ideal para principiantes, pero también cuenta con características avanzadas que lo hacen valioso para desarrolladores experimentados. Para comenzar a aprender Python, es recomendable tener una comprensión básica de los conceptos fundamentales de programación como variables, tipos de datos, estructuras de control (if-else, bucles) y funciones. Además, un conocimiento elemental de matemáticas y lógica será beneficioso para entender y resolver problemas mediante el código.

Para practicar estos conceptos básicos, los estudiantes deben intentar implementar una pequeña calculadora en Python que realice operaciones aritméticas simples como suma, resta, multiplicación y división. El ejercicio debe incluir la definición de funciones para cada operación y la utilización de entradas del usuario.

- Comprender los conceptos fundamentales de programación necesarios.
- Practicar el desarrollo de una calculadora básica en Python.
