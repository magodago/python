Módulo 1: Introduccion y contexto

Como sacar el maximo partido

  • Familiarizarse con las características principales de Python que lo hacen una excelente herramienta para desarrollar aplicaciones de todo tipo.
  • Aprender a utilizar eficazmente las bibliotecas y frameworks disponibles en Python para optimizar el código y acelerar el desarrollo.

Python es un lenguaje de programación de alto nivel, interpretado, orientado a objetos que se destaca por su claridad y simplicidad. Su sintaxis limpia y legible facilita la escritura de programas eficientes y mantenibles, lo que permite a los desarrolladores concentrarse en resolver problemas específicos en lugar de lidiar con detalles complejos del sistema operativo o hardware. Además, Python cuenta con una gran variedad de bibliotecas y frameworks que pueden ser utilizados para automatizar tareas, realizar análisis de datos, crear interfaces gráficas y mucho más.

Para poner en práctica lo aprendido, los estudiantes deberán desarrollar un pequeño proyecto utilizando la biblioteca Pandas para manipular datos. El ejercicio consistirá en cargar un conjunto de datos proporcionado, realizar algunas operaciones básicas como filtrado y agrupación de datos, y finalmente exportar el resultado a un archivo CSV. Este ejercicio no solo permitirá a los estudiantes familiarizarse con la sintaxis básica de Python sino también con una biblioteca muy útil para trabajar con datos.

  • Familiarización con las características principales de Python.
  • Práctica en el uso de la biblioteca Pandas para manipulación de datos.
