# Como sacar el maximo partido

- Comprender los fundamentos de Python y su importancia en la programación moderna.
- Identificar las características que hacen a Python una excelente elección para iniciarse en el mundo del desarrollo de software.

Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Su sintaxis clara y concisa lo hace fácil de aprender y usar, lo que lo convierte en una excelente opción tanto para principiantes como para desarrolladores experimentados. Además, Python cuenta con una gran comunidad de usuarios y una amplia biblioteca estándar, lo cual facilita la resolución de problemas y el desarrollo de aplicaciones complejas.

Para sacar el máximo partido de Python, es fundamental familiarizarse con sus características básicas, como las estructuras de datos (listas, diccionarios, tuplas), los bucles condicionales y cíclicos, así como funciones y módulos. Estos elementos forman la base para construir programas robustos y eficientes.

### Ejercicio
Escribe un programa en Python que solicite al usuario ingresar una lista de números separados por comas, luego imprima el número más grande de la lista. Utiliza listas y funciones para resolver el problema.

### Resumen
- Comprender la importancia y facilidad de aprendizaje de Python.
- Familiarizarse con las estructuras de datos básicas y su uso en programación.
