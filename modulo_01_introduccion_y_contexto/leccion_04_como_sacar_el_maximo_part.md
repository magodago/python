# Como sacar el maximo partido

- Dominar la sintaxis básica de Python.
- Comprender cómo utilizar las funciones y métodos incorporados para optimizar el código.

En esta lección sobre cómo sacar el máximo partido a Python, exploraremos los fundamentos del lenguaje de programación más utilizado en el mundo. Veremos cómo aprovechar las características intrínsecas de Python para escribir código eficiente y legible. Aprenderemos a utilizar funciones y métodos incorporados que no solo simplifican la codificación sino también mejoran la velocidad y eficacia del programa.

Para ilustrar esto, trabajaremos con un ejemplo práctico: la creación de una función que calcula el promedio de una lista de números. Usaremos la función `sum()` para sumar los elementos y `len()` para contar cuántos elementos hay en la lista, lo cual es más eficiente y claro que escribir un bucle manual. Además, aprenderemos sobre listas comprensivas y funciones lambda para manipular datos de manera más concisa.

Ejercicio: Escribe una función que calcule el promedio de una lista de números utilizando `sum()` y `len()`. Luego, refactoriza la función usando una lista comprensiva y una función lambda. Compara los resultados en términos de eficiencia y claridad del código.

- Familiarización con funciones incorporadas.
- Uso de listas comprensivas y funciones lambda para optimizar el código.
