# Que vas a aprender

## Objetivos
- Comprender la importancia y contexto de Python en el desarrollo de software.
- Identificar las principales características que hacen a Python un lenguaje versátil.

## Contenido
Python es un lenguaje de programación de alto nivel, interpretado y de código abierto. Fue creado por Guido van Rossum y lanzado al público en 1991. Es conocido por su sintaxis clara y legible, lo que facilita la escritura de programas limpios y concisos. Python se destaca por ser un lenguaje versátil utilizado en una amplia gama de aplicaciones, desde desarrollo web hasta análisis de datos e inteligencia artificial.

Este curso te introducirá a las bases de Python, explorando su historia, su filosofía y sus aplicaciones más comunes. Aprenderás sobre el entorno de desarrollo integrado (IDE) y cómo instalar Python en tu sistema operativo favorito. Además, se abordarán conceptos fundamentales como variables, tipos de datos básicos y estructuras de control de flujo.

## Ejercicio
Escribe un programa simple que imprima "¡Hola, mundo!" utilizando la sintaxis básica de Python. Luego, modifica el programa para que solicite al usuario su nombre e imprima un saludo personalizado.

```python
# Programa inicial
print("¡Hola, mundo!")

# Ejercicio: Programa con entrada del usuario
nombre = input("Por favor, introduce tu nombre: ")
print(f"¡Hola, {nombre}!")
```

## Resumen
- Python es un lenguaje de programación de alto nivel y código abierto.
- Su sintaxis clara y su versatilidad lo hacen una herramienta valiosa en diversos campos del desarrollo y análisis.
