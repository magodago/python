# Que vas a aprender

## Objetivos
- Comprender los fundamentos de Python y su utilidad en la programación.
- Familiarizarse con el entorno de desarrollo integrado (IDE) para Python.

## Contenido
Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se ha vuelto muy popular debido a su sintaxis clara y legible. Este lenguaje fue creado por Guido van Rossum en 1991 y está diseñado para ser fácil de aprender y usar, lo que lo hace ideal tanto para principiantes como para desarrolladores experimentados. Python se utiliza en una amplia gama de aplicaciones, desde desarrollo web hasta análisis de datos e inteligencia artificial.

En esta lección, nos centraremos en presentar el entorno de desarrollo integrado (IDE) PyCharm, que es un poderoso editor de código que facilita la escritura y depuración del código Python. A través de este IDE, los estudiantes aprenderán a crear proyectos, escribir y ejecutar scripts básicos de Python, así como a manejar errores y debuggear su código.

## Ejercicio
Escribe un programa simple en PyCharm que imprima "¡Hola, mundo!" en la consola. Luego, modifica el programa para que solicite al usuario su nombre y lo salude personalmente.

## Resumen
- Python es un lenguaje de programación fácil de aprender y versátil.
- Se utilizará PyCharm como IDE para escribir y ejecutar código.
- Los estudiantes realizarán un ejercicio básico para familiarizarse con la sintaxis y el flujo de trabajo.
