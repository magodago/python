# Bienvenida al curso

## Objetivos
- Comprender la importancia de Python en el desarrollo de software y ciencia de datos.
- Familiarizarse con el entorno de trabajo de Python.

## Contenido
Bienvenidos al curso de Python. Este lenguaje de programación ha ganado popularidad debido a su simplicidad y versatilidad, lo que lo hace ideal tanto para principiantes como para profesionales en el campo del desarrollo de software y la ciencia de datos. En esta lección, aprenderemos sobre las razones por las que Python es una excelente elección para iniciarse en el mundo de la programación y cómo configurar nuestro entorno de trabajo para comenzar a codificar.

Python se destaca por su sintaxis clara y legible, lo que facilita la escritura de código eficiente y mantenible. Además, cuenta con una gran comunidad de desarrolladores activos que contribuyen constantemente a ampliar su biblioteca estándar y a desarrollar nuevas librerías para diversas aplicaciones.

Para comenzar, es necesario instalar Python en nuestro ordenador. Existen diferentes versiones disponibles, pero la recomendación general es usar la versión más reciente compatible con las herramientas que planeemos utilizar. Una vez instalado, podemos elegir entre varios entornos de desarrollo integrados (IDE) o editores de texto para escribir y ejecutar nuestros programas.

## Ejercicio
Instale Python en su ordenador y descargue un IDE como PyCharm o Visual Studio Code. Cree un archivo nuevo con el nombre "primer_programa.py" y escribe el siguiente código:
```python
print("¡Bienvenido al curso de Python!")
```
Guarda el archivo y ejecútalo para asegurarte de que funciona correctamente.

## Resumen
- Python es un lenguaje de programación versátil ideal para principiantes y profesionales.
- Es importante instalar Python y elegir un IDE adecuado para comenzar a codificar.
- El primer paso en el curso será escribir e ejecutar nuestro primer programa.
