# Bienvenida al curso

Objetivos
- Familiarizar al estudiante con Python como un lenguaje de programación versátil y ampliamente utilizado.
- Introducir los conceptos básicos del entorno de desarrollo e instalación para comenzar a codificar en Python.

Contenido
Python es un lenguaje de programación interpretado, de código abierto y con una sintaxis clara que hace que sea fácil de aprender tanto para principiantes como para profesionales. Su diseño conciso y su gran comunidad lo han convertido en uno de los lenguajes más populares hoy en día, siendo utilizado en diversas áreas como desarrollo web, análisis de datos, inteligencia artificial, ciencia de la computación y mucho más. En este curso, aprenderás a instalar Python en tu sistema operativo preferido, a usar un entorno de desarrollo integrado (IDE) o un editor de texto y a escribir tus primeros programas.

Ejercicio
Instala Python 3 en tu ordenador siguiendo las instrucciones del sitio oficial. Una vez instalado, abre el intérprete interactivo de Python e intenta ejecutar los siguientes comandos:
```python
print("¡Hola, mundo!")
```
Guarda este código en un archivo llamado `hola_mundo.py` y ejecútalo desde la terminal o línea de comandos.

Resumen
- Python es un lenguaje de programación versátil y ampliamente utilizado.
- Es fácil de aprender y se utiliza en diversas áreas como desarrollo web, análisis de datos e inteligencia artificial.
