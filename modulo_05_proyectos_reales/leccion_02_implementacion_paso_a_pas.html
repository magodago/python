Módulo 5: Proyectos reales

Implementacion paso a paso

### Objetivos

  • Implementar un proyecto real utilizando Python que resuelva un problema específico.
  • Familiarizarse con la estructura y buenas prácticas de un proyecto de Python.

### Contenido

En esta lección, se profundizará en el proceso de implementación de proyectos reales utilizando Python. Se comenzará por seleccionar un problema o tarea específica que pueda ser resuelta mediante programación, como la gestión de tareas, análisis de datos o creación de una pequeña aplicación web. Posteriormente, se explorará cómo organizar y estructurar el código en carpetas y archivos para mantenerlo ordenado y fácil de manejar. Se discutirá también la importancia de documentar el código y las prácticas recomendadas para escribir código limpio y legible.

### Ejercicio

Implementa un programa simple que permita al usuario ingresar una lista de tareas diarias y luego muestre estas tareas en orden aleatorio. El programa debe permitir agregar, eliminar o mostrar tareas existentes. Documenta tu código con comentarios descriptivos y utiliza buenas prácticas de nombres de variables y funciones.

### Resumen

  • Seleccionar un problema real para resolver.
  • Organizar el proyecto en carpetas y archivos.
  • Documentar y escribir código limpio.
