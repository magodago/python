# Despliegue

- Dominar la configuración del entorno de producción.
- Implementar el despliegue automático de aplicaciones web con Python.

Para este módulo, aprenderemos a preparar nuestro proyecto para su despliegue en un servidor. Esto incluirá la configuración correcta del archivo `requirements.txt` y el uso de herramientas como Gunicorn o uWSGI para ejecutar nuestra aplicación web. También abordaremos la integración con servicios de hosting como Heroku, utilizando Docker para facilitar este proceso.

Ejercicio: Configura un entorno local de desarrollo utilizando Docker y despliega una aplicación Flask simple en tu máquina. Asegúrate de que todos los requisitos se instalen correctamente a través del archivo `requirements.txt` y que la aplicación funcione sin problemas al ejecutar el contenedor Docker.

Puntos clave:
- Uso adecuado de Docker para facilitar despliegues.
- Configuración correcta de `requirements.txt`.
- Integración con servicios de hosting como Heroku.
