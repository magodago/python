# Diseño del proyecto

Objetivos
- Comprender los conceptos básicos de diseño de proyectos en Python.
- Identificar y planificar las etapas del desarrollo de un proyecto real.

Contenido
En esta lección, se abordarán los fundamentos del diseño de proyectos utilizando Python. Se introducirá el proceso de planificación inicial, donde se identifican los requisitos y objetivos del proyecto. También se explorará la importancia de definir una arquitectura clara y modular que permita la implementación eficiente del software. Se discutirá cómo estructurar el código para facilitar su mantenimiento y escalamiento.

Ejercicio
El estudiante deberá diseñar un proyecto simple en Python, como un sistema de gestión de tareas. Deberá identificar los requisitos funcionales e no funcionales, definir la arquitectura general del proyecto y crear un esbozo de estructura de directorios y archivos.

Resumen
- Identificación clara de requisitos y objetivos.
- Definición de una arquitectura modular para el proyecto.
