Módulo 5: Proyectos reales

Pruebas y validacion

  • Familiarizarse con las herramientas de pruebas y validación en Python.
  • Implementar técnicas básicas de pruebas unitarias para asegurar la calidad del código.

La lección se centra en el uso de herramientas de pruebas y validación en proyectos Python. Se introducirá el concepto de pruebas unitarias utilizando `unittest` y `pytest`, destacando su importancia en el desarrollo de software. Los estudiantes aprenderán a escribir pruebas para comprobar que las funciones del código funcionan como se espera, lo cual es crucial para mantener la integridad del proyecto.

Ejercicio: Implementar pruebas unitarias para una función simple que realiza cálculos matemáticos, utilizando tanto `unittest` como `pytest`. Los estudiantes deberán escribir pruebas para diferentes casos de uso y asegurarse de que su código funcione correctamente en todos ellos.
