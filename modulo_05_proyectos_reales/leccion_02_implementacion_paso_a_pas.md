# Implementacion paso a paso

- Comprender la estructura básica de un proyecto real en Python.
- Implementar una solución a un problema práctico utilizando buenas prácticas de codificación.

En esta lección, se explorará cómo estructurar y desarrollar un proyecto real en Python. Se comenzará con el análisis del problema para identificar las necesidades funcionales y no funcionales. Luego, se diseñará una arquitectura simple que permita la implementación eficiente de la solución. Se utilizará el paradigma orientado a objetos para organizar el código de manera legible y mantenible.

El ejercicio consiste en desarrollar un pequeño sistema de gestión de tareas. Los estudiantes deben crear una clase `Tarea` con atributos como título, descripción y estado (pendiente/completada). A continuación, implementarán métodos para agregar, listar y marcar como completadas las tareas. Se recomienda usar listas en lugar de diccionarios para almacenar las tareas y estructurar el código de manera modular.

- Familiarizarse con la creación de clases y objetos en Python.
- Practicar la implementación de métodos en una clase.
