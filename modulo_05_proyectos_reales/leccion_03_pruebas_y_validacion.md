# Pruebas y validacion

- Comprender la importancia de pruebas y validación en proyectos Python.
- Implementar técnicas básicas de pruebas unitarias utilizando `unittest`.

Python es un lenguaje de programación versátil que se utiliza en una amplia gama de aplicaciones, desde desarrollo web hasta ciencia de datos. Sin embargo, para garantizar el funcionamiento correcto y la calidad del código, es crucial incorporar prácticas de pruebas e integrarlas en el flujo de trabajo de desarrollo. En esta lección, exploraremos cómo utilizar `unittest`, un framework de pruebas incorporado en Python, para escribir y ejecutar pruebas unitarias que verifiquen la funcionalidad individual de los componentes del código.

Para ilustrar este concepto, comenzaremos creando una función simple que realiza cálculos matemáticos. Luego, utilizaremos `unittest` para escribir pruebas que aseguren que esta función funcione correctamente en diferentes escenarios. Este ejercicio no solo ayudará a entender cómo se estructuran las pruebas unitarias sino también cómo se integran en el código existente.

### Ejercicio
1. Crea una función llamada `calcular_area` que tome dos parámetros `base` y `altura` e imprima el área de un rectángulo.
2. Utiliza `unittest` para escribir pruebas unitarias que verifiquen si la función `calcular_area` funciona correctamente en diferentes casos, como cuando se ingresan valores válidos, cero o negativos.

### Resumen
- Pruebas unitarias son esenciales para garantizar el funcionamiento correcto de componentes individuales del código.
- `unittest` proporciona una estructura robusta para escribir y ejecutar pruebas en Python.
