# Pruebas y validacion

- Comprender la importancia de pruebas y validación en el desarrollo de software.
- Implementar técnicas básicas de pruebas unitarias utilizando la biblioteca unittest.

El objetivo principal de esta lección es que los estudiantes comprendan la importancia de las pruebas y la validación en el ciclo de desarrollo de software, especialmente en Python. A través del uso de la biblioteca `unittest`, se aprenderán a escribir pruebas unitarias para verificar que diferentes partes de un programa funcionen como se espera. Las pruebas son cruciales para detectar errores temprano en el proceso de desarrollo y mejorar la calidad del código.

Se introducirá la estructura básica de una prueba unitaria utilizando `unittest.TestCase`, donde los métodos `setUp()` y `tearDown()` se usarán para configurar y limpiar el entorno antes y después de cada prueba. Además, se enseñará cómo utilizar assertions (`assertEqual`, `assertTrue`, etc.) para verificar que las funciones o métodos del código devuelvan o comporten como se espera.

Para practicar, los estudiantes deberán implementar pruebas unitarias para una función simple que realiza cálculos matemáticos. La tarea consiste en escribir pruebas para comprobar la suma, resta, multiplicación y división de dos números enteros utilizando `unittest`.

Los puntos clave a resaltar son:
- El valor crucial de las pruebas en el desarrollo de software.
- Uso práctico de `unittest` para implementar pruebas unitarias.
