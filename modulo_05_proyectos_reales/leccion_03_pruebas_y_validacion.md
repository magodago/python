# Pruebas y validacion

- Comprender la importancia de pruebas y validación en el desarrollo de software.
- Implementar técnicas básicas de pruebas unitarias utilizando la biblioteca unittest de Python.

El objetivo principal de esta lección es que los alumnos comprendan la importancia de incorporar pruebas y validaciones en su código para asegurar su correcto funcionamiento. A través del uso de la biblioteca unittest, aprenderán a escribir pruebas unitarias, lo que les permitirá detectar errores temprano en el desarrollo y mejorar la calidad del software.

En este módulo, se explorará cómo las pruebas pueden ayudar a prevenir errores comunes en aplicaciones Python. Se verá cómo crear test para diferentes tipos de funciones y métodos utilizando assert statements y subclases de unittest.TestCase. Los alumnos aprenderán a ejecutar pruebas de manera manual y automatizada, así como a interpretar los resultados para corregir problemas detectados.

Para practicar, los estudiantes deberán desarrollar un pequeño programa que calcule el área de figuras geométricas (triángulo, círculo y rectángulo) y luego escribir pruebas unitarias para verificar su funcionamiento correctamente. Este ejercicio les permitirá aplicar lo aprendido sobre pruebas y validación en una situación práctica.

- Comprender la importancia de pruebas en el desarrollo de software.
- Implementar pruebas unitarias usando unittest en Python.
