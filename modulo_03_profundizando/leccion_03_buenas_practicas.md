# Buenas practicas

- Mejorar la legibilidad y mantenibilidad del código.
- Aplicar estándares de codificación para Python.

La lección se centrará en las buenas prácticas que los desarrolladores deben seguir para escribir código más limpio, eficiente y fácil de mantener. Se abordarán temas como la importancia de utilizar nombres descriptivos para variables y funciones, la correcta indentación y el uso de espacios, así como la importancia de documentar el código con comentarios claros y precisos.

Para ejemplificar estos conceptos, los estudiantes deberán refactorizar un script existente que realiza cálculos matemáticos básicos. El objetivo es mejorar la legibilidad del código original, añadir documentación adecuada y asegurarse de que cumple con las mejores prácticas recomendadas.

- Mejorar la claridad y legibilidad del código.
- Aplicar estándares de codificación PEP 8.
