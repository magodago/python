# Puntos clave

- Comprender la utilización avanzada de listas y diccionarios.
- Implementar funciones recursivas y comprensión de listas para resolver problemas complejos.

En esta lección profundizaremos en la manipulación de listas y diccionarios, dos estructuras de datos fundamentales en Python. Veremos cómo utilizar comprensiones de listas para simplificar el código y mejorar su eficiencia. Además, exploraremos ejemplos prácticos que demuestran cómo estas herramientas pueden ser aplicadas para resolver problemas complejos. También abordaremos funciones recursivas, discutiendo sus beneficios y desventajas en comparación con las iterativas.

Ejercicio: Dado una lista de números enteros, utiliza comprensión de listas para crear una nueva lista que contenga solo los números pares. Luego, escribe una función recursiva que calcule la suma de todos los elementos de esa lista.

Resumen
- Utilizar comprensiones de listas para simplificar y optimizar el código.
- Implementar funciones recursivas para resolver problemas complejos.
