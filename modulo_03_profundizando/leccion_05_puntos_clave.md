# Puntos clave

- Dominar la utilización de comprensión de listas y diccionarios para simplificar el código.
- Aprender a utilizar decoradores para modificar comportamientos de funciones o métodos.

La lección se enfocará en dos conceptos avanzados de Python: la comprensión de listas y diccionarios, así como los decoradores. La comprensión de listas permite crear listas de manera concisa y eficiente, utilizando una sintaxis más clara que el uso de bucles `for`. Los diccionarios son estructuras de datos clave-valor que permiten almacenar y acceder a datos de manera rápida y flexible. Los decoradores, por otro lado, son funciones que modifican o extienden el comportamiento de otras funciones sin modificar su código original.

Para practicar estos conceptos, los estudiantes deberán escribir un programa que utilice comprensión de listas para filtrar números pares de una lista y luego crear un diccionario con los cuadrados de esos números. Además, se pedirá que implementen un decorador que muestre el tiempo de ejecución de la función.

- Comprender cómo funcionan y aplicar la comprensión de listas y diccionarios.
- Implementar y utilizar decoradores para modificar comportamientos de funciones.
