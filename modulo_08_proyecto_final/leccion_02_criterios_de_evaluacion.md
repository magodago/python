# Criterios de evaluacion

Objetivos
- Comprender los criterios de evaluación para el proyecto final de Python.
- Identificar las áreas a mejorar en la implementación del proyecto.

Contenido
En esta lección se presentarán los criterios de evaluación que se considerarán durante la calificación del proyecto final de Python. Los estudiantes aprenderán sobre la importancia de cumplir con estos criterios para obtener una calificación justa y equitativa. Se abordarán aspectos como la funcionalidad correcta del código, la claridad y legibilidad del mismo, la eficiencia en el uso de recursos, la documentación adecuada y la resolución de problemas.

Ejercicio
Desarrollar un programa Python que implemente un sistema de gestión de tareas. El programa debe permitir al usuario agregar, eliminar y listar tareas pendientes. Además, se evaluará la capacidad del estudiante para documentar el código, explicando cada función y variable utilizada.

Resumen
- Los criterios de evaluación incluyen funcionalidad, claridad, eficiencia, documentación y resolución de problemas.
- El ejercicio propuesto permite aplicar los conocimientos adquiridos en la implementación práctica del proyecto final.
