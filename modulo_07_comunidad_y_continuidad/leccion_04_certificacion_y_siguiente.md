# Certificacion y siguiente nivel

### Objetivos
- Comprender los pasos necesarios para obtener una certificación en Python.
- Explorar las oportunidades de desarrollo profesional que ofrece la obtención de una certificación.

### Contenido
La certificación en Python es un paso crucial para demostrar competencias técnicas y aumentar la valoración del perfil profesional. Existen varias certificaciones reconocidas, como la Certified Associate in Python Programming (CAP) ofrecida por la Python Software Foundation, o el Certified Python DevOps Engineer (CPDE) que combina conocimientos de Python con habilidades en DevOps. Para obtener una certificación, es importante familiarizarse con los contenidos cubiertos en ella, realizar ejercicios y exámenes preparatorios, y estar al tanto de las últimas actualizaciones en la comunidad Python. Además, participar activamente en foros y comunidades de desarrolladores puede mejorar significativamente las posibilidades de éxito.

### Ejercicio
Realiza el examen de certificación CAP (Certified Associate in Python Programming) disponible en el sitio web oficial de la Python Software Foundation. Antes de tomar el examen, revisa los temas cubiertos en el examen y completa ejercicios prácticos relacionados con esos temas.

### Resumen
- La obtención de una certificación en Python puede mejorar significativamente las oportunidades laborales.
- Es importante familiarizarse con la estructura del examen antes de presentarlo.
- Participar activamente en comunidades y foros de Python puede ser beneficioso para el proceso de preparación.
