# Recursos de la comunidad

- Familiarizarse con los recursos de la comunidad Python disponibles.
- Identificar y explotar las oportunidades de aprendizaje y colaboración proporcionadas por la comunidad.

La lección se centrará en explorar cómo la comunidad Python ofrece una amplia gama de recursos para el desarrollo, el aprendizaje y la colaboración. Se destacarán plataformas como GitHub, donde los estudiantes podrán encontrar códigos de ejemplo, contribuir a proyectos o iniciar sus propios repositorios. Además, se discutirá sobre foros y grupos de discusión en línea, tales como Stack Overflow y Reddit, que son excelentes para resolver dudas y compartir conocimientos. Se explicará cómo utilizar herramientas como PyPI (Python Package Index) para instalar paquetes y bibliotecas útiles.

Ejercicio: Los estudiantes deben buscar y contribuir a un proyecto de código abierto en GitHub. Deben seleccionar un proyecto relevante, leer la documentación disponible, realizar una pequeña modificación o corrección y enviar un pull request para su revisión.

- Comprender el valor de los recursos comunitarios.
- Practicar la colaboración y contribución a proyectos de código abierto.
