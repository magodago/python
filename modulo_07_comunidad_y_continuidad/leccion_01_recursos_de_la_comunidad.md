# Recursos de la comunidad

- Familiarizarse con los recursos de la comunidad Python disponibles.
- Aprender a utilizar y contribuir a plataformas de apoyo comunitario para mejorar habilidades en Python.

La lección se centrará en explorar los diversos recursos que ofrece la comunidad Python. Se destacarán plataformas como Stack Overflow, GitHub y el sitio oficial de Python, donde se pueden encontrar tutoriales, ejemplos de código, documentación y foros de discusión. Estos recursos son valiosos para resolver problemas, aprender nuevas técnicas y mantenerse actualizado con las últimas novedades en la comunidad.

Ejercicio: El estudiante debe buscar y seleccionar tres recursos útiles dentro de la comunidad Python (una plataforma, un proyecto en GitHub y una guía de documentación) relacionados con temas que le interesen. Debe describir brevemente cada recurso y cómo puede ser útil para su aprendizaje.
