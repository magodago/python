# Actualizaciones

- Los estudiantes aprenderán sobre las mejores prácticas para seguir actualizaciones en Python.
- Comprenderán cómo mantener sus proyectos y entorno de desarrollo actualizados.

Python es un lenguaje de programación dinámicamente tipado que se renueva constantemente con nuevas versiones y mejoras. Es importante para los desarrolladores mantenerte al tanto de estas actualizaciones, ya que pueden incluir mejoras en rendimiento, correcciones de errores y nuevas funcionalidades que podrían ser útiles para tu trabajo o proyectos futuros.

Para mantenerse actualizado, se recomienda a los estudiantes suscribirse a listas de correo electrónico de Python, como el grupo de discusión oficial de Python (python-announce@python.org), y seguir las cuentas oficiales de Python en redes sociales. Además, es recomendable leer regularmente la documentación oficial de Python, que se actualiza con cada nueva versión.

### Ejercicio
Realiza una búsqueda de las últimas actualizaciones de Python en el sitio web oficial (https://www.python.org). Identifica al menos tres novedades relevantes y explícalas brevemente. Luego, comparte tus hallazgos en un foro de discusión del curso.

- Los estudiantes identificarán las últimas actualizaciones de Python.
- Comprenderán la importancia de mantenerse informado sobre nuevas versiones y mejoras.
